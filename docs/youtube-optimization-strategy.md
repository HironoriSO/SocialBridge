### 2-3. タイトル・サムネイル最適化ルール

#### タイトル
- **文字数**: 28-40文字（スマホで見切れない長さ）
- **構造**: 【カテゴリ】+メインKW+ベネフィット
- **例**: 「【MEO対策】産廃業者がGoogleマップで1位になる方法｜問い合わせ3倍」
- **禁止**: 釣りタイトル、過度な煽り（業界の信頼を損なう）

#### サムネイル
- **文字**: 最大13文字、太ゴシック体
- **配色**: ブランドカラー（青系）+ アクセント（黄色/赤）
- **人物**: 代表の宋さんの顔を入れる（信頼性UP）
- **統一感**: シリーズごとにテンプレートを作成

## 4. 問い合わせ導線の設計

### 4-1. YouTube → 問い合わせの流れ

```
[認知] YouTube動画（Shorts/検索流入）
  ↓
[興味] 長尺動画で深い情報提供 + チャンネル登録
  ↓
[検討] 概要欄リンク → サービスページ (social-bridge.net/dx_meo/)
  ↓
[行動] 無料相談申し込み / お問い合わせフォーム
```

### 4-2. 各動画の CTA パターン

| タイミング | CTA内容 |
|-----------|---------|
| 動画冒頭（30秒） | 「詳しくは概要欄のリンクから無料相談もできます」 |
| 動画中盤 | 「こういった施策を御社でも導入したい方は概要欄をチェック」 |
| 動画終盤 | 「無料相談受付中！概要欄の最初のリンクからお申し込みください」 |
| 固定コメント | サービスページURLと簡潔な訴求文 |
